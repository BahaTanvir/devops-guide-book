## What You'll Learn

Sarah accidentally commits an API key to Git, and now it's public. By the end of this chapter, you'll know how to:

- Use dedicated secrets management tools (HashiCorp Vault, AWS Secrets Manager, sealed-secrets)
- Implement secrets rotation policies and automation
- Detect and prevent secrets from being committed to version control
- Respond to secrets exposure incidents (revoke, rotate, audit)
- Design applications that fetch secrets at runtime, not build time
- Apply the principle of least privilege to secrets access

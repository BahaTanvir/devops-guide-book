# Chapter 23: The Security Scare

> *"Security vulnerabilities don't wait for convenient times to be discovered."*

## What You'll Learn

A critical CVE is discovered in a dependency Sarah's team uses everywhere. By the end of this chapter, you'll know how to:

- Implement container image scanning in CI/CD pipelines
- Use vulnerability databases (CVE, NVD) to track security issues
- Establish a patch management process for dependencies and base images
- Configure security policies with tools like OPA (Open Policy Agent)
- Implement runtime security monitoring to detect malicious behavior
- Balance security urgency with deployment stability and testing

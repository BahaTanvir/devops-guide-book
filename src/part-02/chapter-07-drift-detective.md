# Chapter 7: The Drift Detective

> *"Infrastructure drift is technical debt compounding with interest."*

---

## What You'll Learn

Someone manually modified production infrastructure, causing mysterious drift from Terraform's expected state. By the end of this chapter, you'll know how to:

- Identify the causes and consequences of infrastructure drift
- Use `terraform plan` and `terraform refresh` to detect configuration divergence
- Implement drift detection automation with scheduled Terraform runs
- Decide when to import existing resources vs. recreating them
- Design policies and guardrails that prevent manual changes
- Set up alerts and dashboards to monitor infrastructure compliance

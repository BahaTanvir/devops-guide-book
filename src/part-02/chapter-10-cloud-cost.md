## What You'll Learn

The monthly cloud bill suddenly triples, and Sarah needs to find the culprit. By the end of this chapter, you'll know how to:

- Set up cost budgets, alerts, and forecasting
- Design infrastructure with cost efficiency in mind (right-sizing, reserved instances, spot instances)
- Use tools like Infracost to estimate Terraform changes before deployment
